# www

Sitio web de prueba

## Installation

Use the package manager [prueba](https://prueba/www/) to install foobar.

```bash
pip install www
```

## Usage

```python
import www

www.pluralize('word') # returns 'words'
www.pluralize('goose') # returns 'geese'
www.singularize('phenomena') # returns 'phenomenon'
```
